# CoworkAny 0.1.0-beta.1

## Purpose
- Small external beta release for controlled testers.
- Focus on validating desktop packaging, runtime stability, and core assistant flows.

## Branch And Tag
- Branch: `release/0.1.0-beta.1`
- Recommended tag: `v0.1.0-beta.1`
- GitHub release mode: `prerelease: true`

## Included Hardening
- Tauri core and CLI pinned to a stable v2.10 toolchain.
- Node runtime aligned to `22.22.1` via `.nvmrc` and workflow setup.
- Desktop CI widened to the full acceptance suite instead of a single smoke file.
- Tauri CSP is explicit instead of disabled.
- RAG sidecar startup no longer kills arbitrary processes on port `8787`.

## Validation Snapshot
- `desktop`: `npm test`
- `desktop`: `npm run build`
- `desktop/src-tauri`: `cargo check --locked`
- `sidecar`: `npm run typecheck`

## Beta Operator Checklist
1. Ensure GitHub secrets for Tauri signing are present.
2. Use tag `v0.1.0-beta.1` or run the `Release` workflow manually with `prerelease` enabled.
3. Archive the generated desktop bundles from the workflow artifacts.
4. Distribute only to named beta testers with install and rollback instructions.

## Known Limits
- Windows code signing and macOS notarization are still not covered here.
- The working tree contains additional in-flight product work outside this beta packaging pass.
- Manual desktop smoke on packaged binaries is still recommended before sharing externally.
