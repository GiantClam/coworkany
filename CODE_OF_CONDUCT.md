# Code of Conduct

## Our Standard

CoworkAny is an engineering-focused project. We expect contributors to be respectful, direct, and constructive.

Examples of behavior that help the project:

- giving technically grounded feedback
- asking clear questions
- documenting assumptions and tradeoffs
- disagreeing on ideas without attacking people
- reporting problems with enough detail to reproduce them

Examples of unacceptable behavior:

- harassment or personal attacks
- abusive or insulting language
- deliberate misrepresentation of another contributor's intent
- sharing private information without permission
- disruptive behavior that makes collaboration harder for others

## Scope

This code of conduct applies to:

- GitHub issues
- pull requests
- discussions
- release notes and documentation contributions

## Enforcement

Project maintainers may remove or edit comments, close threads, or block participation when behavior violates these standards.

If you experience or witness unacceptable behavior, contact the maintainers privately before escalating publicly when possible.

## Interpretation

This document is intentionally short. When in doubt, prefer professionalism, clarity, and respect.
